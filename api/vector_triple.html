Vector Triple

template<typename VectorT>
class VectorTriple

Class that packs sets of 3 coordinates into a single unit for operations on interleaved or deinterleaved data. The number of coordinates held is determined by the SIMD width. If constructing a SIMD typed VectorTriple from a scalar array, the requisite deinterleave operations are performed in the constructor.

tparam VectorT

(SIMD datatype) or scalar (float or double) type
